# Atelier Nova - Layout Premium

Layout premium com paginas prontas (home, projetos, servicos, sobre, contato e blog).

## Estrutura
- `index.html` (home)
- `window/` (paginas internas)
- `assets/css` (estilos)
- `assets/jv` (scripts)
- `assets/img` (imagens)
- `assets/favicon.svg` (favicon)

## Como visualizar
- Abra `index.html` no navegador ou use uma extensao tipo Live Server.

## Publicar no GitHub Pages
1. Crie o repositorio e envie os arquivos.
2. Em Settings > Pages, selecione a branch `main` e a pasta `/ (root)`.
3. Aguarde o link do GitHub Pages.

### Ajustes obrigatorios antes de publicar
- Troque os placeholders do OG/Twitter e do GitHub:
  - `https://maicon-oliveira-dev.github.io/neon-ui-assets/`
  - `https://github.com/maicon-oliveira-dev/neon-ui-assets`
